# GyeongHoKim Tap

## How do I install these formulae?

`brew install gyeonghokim/tap/<formula>`

Or `brew tap gyeonghokim/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "gyeonghokim/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
